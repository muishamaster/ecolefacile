Questions fréquentes

Trouvez rapidement les réponses aux questions les plus posées par les parents

Inscription

Comment inscrire mon enfant sur EcoleFacile.cd ?

L'inscription se fait en 3 étapes simples :

  1. Choisissez une école dans notre liste d'établissements partenaires
  2. Remplissez le formulaire avec les informations de votre enfant
  3. Téléchargez les documents requis (acte de naissance, photo)
  4. Confirmez et payez les frais d'inscription

L'inscription prend environ 10 minutes. Un numéro de confirmation vous sera envoyé par SMS.

Puis-je inscrire plusieurs enfants en même temps ?

Oui, absolument ! Pour inscrire plusieurs enfants :

  1. Après avoir inscrit le premier enfant, vous verrez un bouton "Ajouter un autre enfant"
  2. Cliquez dessus pour recommencer le processus pour le deuxième enfant
  3. Vous pouvez inscrire jusqu'à 5 enfants en une seule session

Les frais seront calculés séparément pour chaque enfant selon l'école choisie.

Y a-t-il un âge minimum ou maximum pour l'inscription ?

L'âge requis dépend de chaque école et du niveau scolaire. En général :

  • Maternelle : 3-5 ans
  • Primaire : 6-11 ans
  • Secondaire : 12-18 ans

Vérifiez la page de l'école pour les conditions spécifiques. Certaines écoles peuvent avoir des tests d'admission pour certains niveaux.

Que faire si l'école que je cherche n'est pas dans la liste ?

Si votre école n'est pas encore partenaire, vous pouvez :

  • Nous contacter via le formulaire de contact ou WhatsApp
  • Proposer à l'école de nous rejoindre (c'est gratuit pour les écoles)
  • Vérifier régulièrement car nous ajoutons de nouvelles écoles chaque semaine

Vous pouvez aussi nous envoyer le nom de l'école, nous la contacterons pour l'ajouter.

Paiement

Quels sont les moyens de paiement acceptés ?

Nous acceptons plusieurs moyens de paiement adaptés à la RDC :

  • M-Pesa - Paiement instantané
  • Airtel Money - Paiement instantané
  • Orange Money - Paiement instantané
  • Virement bancaire - Traitement sous 24-48h
  • Paiement en agence - Code généré à présenter à la banque

Les paiements Mobile Money sont instantanés et sécurisés.

Puis-je payer en plusieurs fois ?

Cela dépend des écoles :

  • Certaines écoles proposent le paiement en 2 ou 3 fois
  • L'option est indiquée sur la page de l'école et lors du paiement
  • Contactez l'école directement pour discuter d'un arrangement personnalisé

Si l'option de paiement échelonné est disponible, vous verrez "Payer en plusieurs fois" lors du choix du mode de paiement.

Que faire si mon paiement échoue ?

En cas d'échec de paiement :

  1. Vérifiez votre solde Mobile Money
  2. Assurez-vous que votre numéro est correct
  3. Réessayez après 5-10 minutes (parfois des délais réseau)
  4. Si le problème persiste, contactez votre opérateur :
    • M-Pesa : *555#
    • Airtel Money : *125#
    • Orange Money : *144#

Important : Si l'argent a été débité mais que l'inscription n'est pas confirmée, contactez-nous immédiatement avec le numéro de transaction. Nous régulariserons sous 24h.

Les prix sont-ils en dollars ou en francs congolais ?

Les prix sont affichés dans la devise choisie par l'école :

  • La plupart des écoles privées affichent leurs tarifs en dollars ($)
  • Les écoles publiques affichent généralement en francs congolais (FC)

Au moment du paiement, vous verrez le montant converti si vous payez dans une autre devise. Le taux de change appliqué est celui du jour de la transaction.

Documents

Quels documents sont nécessaires pour l'inscription ?

Les documents généralement requis sont :

  • Acte de naissance de l'enfant (photo ou scan)
  • Photo d'identité récente de l'enfant
  • Certificat médical (selon l'école)
  • Bulletins de l'année précédente (pour les transferts)
  • Photo du parent (parfois demandée)

Formats acceptés : PDF, JPG, PNG. Taille maximale : 5 Mo par fichier.

Comment prendre une bonne photo avec mon téléphone ?

Conseils pour prendre une photo acceptable :

  1. Bonne luminosité - Évitez le flash, préférez la lumière naturelle
  2. Cadrage - L'enfant doit occuper 70% de la photo
  3. Fond clair - De préférence blanc ou uni
  4. Visage bien visible - Pas de chapeau ou de lunettes de soleil

Pour les documents (acte de naissance), prenez la photo à plat, bien éclairée, sans ombre.

Puis-je modifier les documents après l'inscription ?

Une fois l'inscription validée, vous ne pouvez plus modifier les documents vous-même. Si vous devez remplacer un document :

  1. Contactez le support via WhatsApp ou email
  2. Indiquez votre numéro de confirmation et le document à remplacer
  3. Envoyez le nouveau document (nous vous donnerons la procédure)

Les modifications sont traitées sous 24-48h.

Écoles

Comment sont sélectionnées les écoles sur la plateforme ?

Nos écoles partenaires :

  • Sont officiellement reconnues par l'État congolais
  • Ont fait l'objet d'une vérification de leurs installations
  • Respectent des standards de qualité d'enseignement
  • Sont recommandées par les parents et la communauté

Nous vérifions chaque école avant de l'ajouter à notre plateforme. Vous pouvez nous signaler une école qui mériterait d'être ajoutée.

Puis-je visiter l'école avant d'inscrire mon enfant ?

Oui, absolument ! Nous vous encourageons même à le faire. Sur la page de chaque école, vous trouverez :

  • Les coordonnées pour prendre rendez-vous
  • Les horaires de visite généralement proposés
  • Un bouton "Contacter l'école" pour envoyer un message direct

La visite est essentielle pour vérifier que l'environnement convient à votre enfant.

Les informations sur les frais sont-elles à jour ?

Oui, nous nous engageons à :

  • Mettre à jour les tarifs chaque année scolaire
  • Vérifier avec les écoles avant la rentrée
  • Signaler clairement tout changement en cours d'année

Si vous constatez une différence entre nos tarifs et ceux annoncés par l'école, signalez-le nous immédiatement pour que nous vérifiions.

Technique

Le site ne se charge pas correctement, que faire ?

Si vous rencontrez des problèmes de chargement :

  1. Vérifiez votre connexion internet
  2. Rafraîchissez la page (F5 ou tirez vers le bas sur mobile)
  3. Videz le cache de votre navigateur
  4. Essayez avec un autre navigateur (Chrome, Firefox, Safari)
  5. Réduisez la qualité des images si vous êtes en connexion lente

Notre site est optimisé pour fonctionner même avec des connexions 3G/4G en RDC.

Est-ce que je peux utiliser le site sans internet ?

Notre site est conçu comme une Progressive Web App (PWA) :

  • Une fois visité, certaines pages restent accessibles hors ligne
  • Les informations de base sur les écoles sont mises en cache
  • Pour l'inscription et le paiement, une connexion internet est nécessaire

Sur Android, vous pouvez "Ajouter à l'écran d'accueil" pour une expérience proche d'une application native.

Mes données sont-elles sécurisées ?

La sécurité de vos données est notre priorité :

  • Chiffrement SSL - Toutes les communications sont cryptées
  • Paiements sécurisés - Nous ne stockons pas vos informations de paiement
  • Hébergement local - Vos données restent en RDC
  • Accès restreint - Seules les personnes autorisées peuvent consulter vos informations

Consultez notre politique de confidentialité pour plus de détails.
